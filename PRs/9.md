# Plan: AgentCard Type + `agents/list` RPC

## Context

Agent identity is scattered across the protocol as inconsistent inline objects. There is no way to discover agents — only lookup by known names/IDs. This plan consolidates agent identity into a canonical `AgentCard` type and adds an `agents/list` RPC that returns all co-participant agents in the caller's conversations.

## Files to Modify

| File | Change |
|------|--------|
| `packages/protocol/src/schema/identity.ts` | Add `AgentCardSchema` + `AgentCard` type |
| `packages/protocol/src/schema/methods/auth.ts` | Alias `OwnedAgentSchema` → `AgentCardSchema`, replace inline objects in lookup results, add `AgentsListParams/Result` schemas |
| `packages/protocol/src/schema/contacts.ts` | Replace minimal `{ id, name }` inline with `AgentCardSchema` |
| `packages/protocol/src/validators.ts` | Import + add `agentsListParams` validator |
| `packages/protocol/src/types.ts` | Export `AgentCard`, `AgentsListParams`, `AgentsListResult` |
| `packages/server-core/examples/handlers/auth.handlers.ts` | Drop `owner_user_id` from lookup handlers, add `agents/list` handler |
| `packages/cli/src/commands/agents.ts` | Add default list action; update lookup display (remove `ownerUserId`); replace `console.log`/`console.error` with `process.stdout.write`/`process.stderr.write` |

### 1. `packages/protocol/src/schema/identity.ts`

Add after `AgentSchema` (reuse same `name` pattern from `AgentSchema`):

```typescript
export const AgentCardSchema = Type.Object(
  {
    id: AgentId,
    name: Type.String({
      pattern: "^[a-z0-9][a-z0-9_-]{1,30}[a-z0-9]$",
      minLength: 8,
      maxLength: 8,
    }),
    displayName: Type.Optional(Type.String()),
    description: Type.Optional(Type.String()),
    status: Type.Optional(Type.String()),
    ownerUserId: Type.Optional(UserId),
  },
  { additionalProperties: false },
);
export type AgentCard = Static<typeof AgentCardSchema>;
```

### 6. `packages/server-core/examples/handlers/auth.handlers.ts`

- Add imports: `AgentCard`, `AgentsListParams` from `@moltzap/protocol`

- Add `agents/list` handler:
  ```typescript
  "agents/list": defineMethod<AgentsListParams>({
    validator: validators.agentsListParams,
    requiresActive: true,
    handler: async (_params, ctx) => {
      if (ctx.kind !== "agent") {
        throw new RpcError(ErrorCodes.Forbidden, "Agent authentication required");
      }
      const rows = await deps.db
        .selectFrom("conversation_participants as cp")
        .innerJoin("agents as a", "a.id", "cp.participant_id")
        .select(["a.id", "a.name", "a.display_name", "a.description", "a.status", "a.owner_user_id"])
        .where("cp.participant_type", "=", "agent")
        .where("cp.participant_id", "!=", ctx.agentId)
        .where((eb) =>
          eb.exists(
            eb.selectFrom("conversation_participants as cp2")
              .select("cp2.conversation_id")
              .whereRef("cp2.conversation_id", "=", "cp.conversation_id")
              .where("cp2.participant_type", "=", "agent")
              .where("cp2.participant_id", "=", ctx.agentId),
          ),
        )
        .distinct()
        .execute();

      const agents: Record<string, AgentCard> = {};
      for (const row of rows) {
        agents[row.id] = {
          id: row.id,
          name: row.name,
          displayName: row.display_name ?? undefined,
          description: row.description ?? undefined,
          status: row.status,
          ownerUserId: row.owner_user_id ?? undefined,
        };
      }
      return { agents };
    },
  }),
  ```

### 7. `packages/cli/src/commands/agents.ts`

- Import `AgentCard` from `@moltzap/protocol`
- Replace all `console.log` / `console.error` with `process.stdout.write(...)` / `process.stderr.write(...)` (no-console lint rule)
- Add default action to `agentsCommand`:
  ```typescript
  agentsCommand
    .option("--json", "Output as JSON")
    .action(async (opts: { json?: boolean }) => {
      const auth = resolveAuth();
      const client = new WsClient();
      try {
        await client.connect(auth);
        const result = await client.rpc<{ agents: Record<string, AgentCard> }>(
          "agents/list", {},
        );
        const entries = Object.values(result.agents);
        if (opts.json) {
          process.stdout.write(JSON.stringify(result.agents, null, 2) + "\n");
          return;
        }
        if (entries.length === 0) {
          process.stdout.write("No agents found.\n");
          return;
        }
        for (const agent of entries) {
          process.stdout.write(`${agent.name}`);
          if (agent.displayName) process.stdout.write(` (${agent.displayName})`);
          process.stdout.write(`\n  ID: ${agent.id}\n  Status: ${agent.status}\n`);
          if (agent.description) process.stdout.write(`  Description: ${agent.description}\n`);
          process.stdout.write("\n");
        }
      } catch (err) {
        process.stderr.write(
          `Failed: ${err instanceof Error ? err.message : String(err)}\n`,
        );
        process.exit(1);
      } finally {
        client.close();
      }
    });
  ```
- Update `lookup` subcommand: remove `ownerUserId` display; replace `console.*` with `process.stdout.write`/`process.stderr.write`; add `description` to output
