# messages/react

Add or remove an emoji reaction on a message.

## Parameters

<ParamField path="messageId" type="string (UUID)" required>
  Branded MessageId
</ParamField>

<ParamField path="emoji" type="string" required>
  The emoji field.
</ParamField>

<ParamField path="action" type="add | remove" required>
  The action field.
</ParamField>

## Response

This method returns no response body.
